# CRUD – Cadastro de Pessoas

Projeto backend simples desenvolvido em Node.js com Express.

Este projeto foi criado com foco didático, com o objetivo de demonstrar:
- a construção de uma API REST básica
- o uso do Git e do GitHub como ferramentas de versionamento durante o desenvolvimento

---

## Tecnologias utilizadas
- Node.js
- Express
- Git e GitHub
- Vite (front-end)

---

## Estrutura do projeto
A aplicação segue uma estrutura simples:

- O Express é responsável por iniciar o servidor
- A API expõe rotas REST para manipulação de dados
- Os dados são armazenados em memória, apenas para fins de estudo

---

## Rotas disponíveis

`POST /usuarios`
`GET /usuarios`

---

## Testes da API
As rotas podem ser testadas utilizando ferramentas de requisições HTTP, como:
- Thunder Client
- Insomnia

---

## Versionamento
O projeto utiliza Git para controle de versão, com commits semânticos, como:

- `feat`: inclusão de novas funcionalidades
- `fix`: correções realizadas durante o desenvolvimento
- `docs`: atualização de documentação

---

## Uso educacional
Este repositório pode ser utilizado em contextos educacionais para demonstrar:

- Desenvolvimento incremental de software
- Funcionamento de uma API REST
- Importância do controle de versão
- Leitura e compreensão do histórico de commits
